
# DesiTech Solutions GitHub Organization

Welcome to the official GitHub organization for **DesiTech Solutions**!  
We build modern, scalable, and reliable software products for Indian businesses, covering domains like retail, service management, accounting, healthcare and so on.
Our flagship products include **Vyaparsathi** and **VyaparMitra, HealthCareMagement**, each with dedicated frontend and backend repositories.

---


## 🛠️ Tech Stack

**Frontend:**  
- Angular  
- React  
- TypeScript
- Next.js  

**Backend:**  
- Spring Boot (3.5.5)  
- Java 17  
- Spring Security + JWT  
- Spring Data JPA  
- Kafka (event-driven microservices)  
- MySQL  
- Flyway (database migrations)  
- MapStruct (DTO mapping)  
- Lombok (boilerplate reduction)

---

## 🔒 Security & Best Practices

- JWT-based authentication for all APIs  
- DTO mapping via MapStruct for clean entity-DTO conversion  
- Database migrations managed with Flyway  
- Event-driven architecture with Kafka  
- Reduced boilerplate code with Lombok
